# Polaris Console

Visit [website](https://polarismesh.cn) to learn more

## Getting started

### Preparing environment

Node 开发环境

Go 1.12 及以上版本，本项目依赖 go mod 进行包管理

### Build

```
git clone https://github.com/PolarisMesh/polaris-console.git
```

```
./build.sh
```

### 提交依赖

将 Go 依赖包复制到 vendor

```
go mod vendor
```
